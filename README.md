# Price Prediction ML Project

This project predicts vehicle selling prices, classifies clients, and segments them using machine learning. It also provides a dashboard for data exploration with interactive visualizations of Rwanda's districts.

## Features

- **Regression:** Predict vehicle selling price based on year, kilometers driven, seating capacity, and estimated income.
- **Classification:** Classify clients into categories.
- **Clustering:** Segment clients into groups like Economy, Standard, and Premium using KMeans.
- **Data Exploration:** View dataset and visualize vehicle clients per district on a Rwanda map.
